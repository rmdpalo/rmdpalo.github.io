Flabby Burp

A flappy bird clone made in Unity.

Role

Developer

What I did

A micro game built on Unity using C# following a tutorial. This is my first exposure to both of these things. Learned about: GameObjects, Events, C# syntax, TextMeshPro UI Elements, Legacy UI Elements, and more. Expanded upon the foundation of the tutorial by learning how to create multiple scenes, adding additional edge cases to prevent game breaking bugs, and hosting online so that people can play the game.
