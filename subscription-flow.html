Subscription Flow

Tasked with creating a "quiz"-like app to try and draw new customers using a different ordering experience.

Role

Developer

What I did

Developed a static page that simulates multiple "pages" and stores a customer's choices so they can go back and edit even if they go to a different URL.
It is comprised of 3 main components: the progress bar, the container of the options, and the controls.

This was created as part of an a/b test where traffic was directed through to this page to measure performance between this experience and the current one we had live at the time.
